# ArchitectGen Libraries

This directory is generated by ArchitectGen and can be consumed as a reusable CMake library workspace.

## Included Domain Libraries

- ArchitectGen::Data_Persistence

## Use In Another Project

### Option 1: add_subdirectory

```cmake
add_subdirectory(path/to/generated/templates external/architectgen)
target_link_libraries(MyApp PRIVATE ArchitectGen::Data_Persistence)
```

### Option 2: install and find_package

```powershell
cmake -S path/to/generated/templates -B build
cmake --build build
cmake --install build --prefix path/to/install
```

```cmake
find_package(ArchitectGenLibraries CONFIG REQUIRED PATHS path/to/install)
target_link_libraries(MyApp PRIVATE ArchitectGen::Data_Persistence)
```
